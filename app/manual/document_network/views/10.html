Request

Click "Add New Draft" and a pop-up form will appear.



On Document Type, Select "request".



Fill-up the form.



Select what type of request you are creating. For request type that is not in the list, you can simply type it in the input field.




Keywords are important for instant search engine. Type any words related to the topic and hit "Enter".

After completing the form. The "Create Draft" button will now be available.
